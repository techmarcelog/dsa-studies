# 🗂️ HashMap (Tabela Hash)

## O que é?

Um **HashMap** é uma estrutura de dados que armazena pares de **chave → valor**, permitindo acesso, inserção e remoção em tempo constante.

> Em vez de buscar por índice numérico como no Array, você busca por uma **chave significativa** — como uma palavra, ID ou qualquer outro identificador.

---

## Como funciona?

Internamente, o HashMap usa uma **função hash** que transforma a chave em um índice numérico para saber onde guardar o valor na memória:

```
chave: "nome"
         |
   [ hash function ]
         |
       índice: 3
         |
[ _ | _ | _ | "Marcelo" | _ | _ ]
               posição 3
```

```
Inserindo:  put("nome", "Marcelo")  →  hash("nome") = 3  →  guarda na posição 3
Buscando:   get("nome")            →  hash("nome") = 3  →  retorna "Marcelo"
```

---

## Estrutura

```
HashMap {
  "nome"  → "Marcelo"
  "idade" → 22
  "lang"  → "Java"
}
```

| Componente | Descrição |
|------------|-----------|
| **Chave (Key)**   | Identificador único do dado |
| **Valor (Value)** | Dado armazenado |
| **Hash Function** | Converte a chave em um índice numérico |

---

## Complexidade

| Operação  | Complexidade |
|-----------|-------------|
| Acesso    | O(1) médio  |
| Inserção  | O(1) médio  |
| Remoção   | O(1) médio  |

> O(1) porque a hash function calcula diretamente onde o dado está — sem percorrer nada.

---

## Colisão

Ocorre quando dois chaves diferentes geram o **mesmo índice** após o hash:

```
hash("nome")  = 3
hash("apelido") = 3  ← colisão!
```

O HashMap resolve isso internamente (encadeamento ou endereçamento aberto), mas colisões frequentes podem degradar a performance para **O(n)** no pior caso.

---

## Comparação com Array

| Estrutura   | Acesso por    | Complexidade |
|-------------|---------------|-------------|
| **Array**   | Índice numérico | O(1)       |
| **HashMap** | Chave (qualquer tipo) | O(1) médio |

---

## Resumo

- 🔑 Armazena dados em pares **chave → valor**
- ⚡ Acesso em **O(1)** na maioria dos casos
- ⚠️ **Colisões** podem ocorrer e impactar a performance
- ✅ Ideal para: contagem de frequências, cache, índices, dicionários
