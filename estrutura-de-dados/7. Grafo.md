# 🕸️ Grafo (Graph)

## O que é?

Um **Grafo** é uma estrutura de dados que representa **relacionamentos entre elementos**. É formado por **vértices (nós)** conectados por **arestas (edges)**.

> Diferente da Binary Tree, um grafo não tem hierarquia — qualquer nó pode se conectar a qualquer outro.

---

## Estrutura

```
    (A) ——— (B)
     |     / |
     |    /  |
    (C) ——  (D)
         \  /
          (E)
```

| Componente | Descrição |
|------------|-----------|
| **Vértice (Node)** | Elemento do grafo |
| **Aresta (Edge)**  | Conexão entre dois vértices |

---

## Tipos de grafo

### Direcionado vs Não-direcionado

```
Não-direcionado:        Direcionado:
  A ——— B                A ——→ B
  (vai e volta)          (só vai)
```

| Tipo | Descrição |
|------|-----------|
| **Não-direcionado** | A conexão vale nos dois sentidos |
| **Direcionado (Dígrafo)** | A conexão tem sentido definido |

### Com peso vs Sem peso

```
Com peso:               Sem peso:
  A —5— B                 A ——— B
  A —2— C                 A ——— C
  (custo da aresta)
```

| Tipo | Descrição |
|------|-----------|
| **Com peso** | Cada aresta tem um valor (distância, custo, tempo) |
| **Sem peso** | Conexão simples, sem valor associado |

---

## Representações na memória

### Lista de adjacência
```
A → [B, C]
B → [A, D, E]
C → [A, E]
```
> Mais eficiente para grafos esparsos (poucos edges)

### Matriz de adjacência
```
    A  B  C  D
A [ 0, 1, 1, 0 ]
B [ 1, 0, 0, 1 ]
C [ 1, 0, 0, 0 ]
D [ 0, 1, 0, 0 ]
```
> Mais eficiente para grafos densos (muitos edges)

---

## Comparação com outras estruturas

| Estrutura | Relação entre nós |
|-----------|------------------|
| **Linked List** | Linear (1 → 1) |
| **Binary Tree** | Hierárquica (1 → 2) |
| **Grafo**       | Livre (N → N) |

---

## Casos de uso reais

| Aplicação | Exemplo |
|-----------|---------|
| Redes sociais | Usuários e conexões entre eles |
| GPS / Mapas | Cidades e estradas com distâncias |
| Internet | Servidores e conexões de rede |
| Recomendações | Relação entre produtos ou conteúdos |

---

## Resumo

- 🔵 **Vértices** são os nós, **arestas** são as conexões
- 🔀 Pode ser **direcionado** ou **não-direcionado**
- ⚖️ Pode ter **peso** nas arestas ou não
- 🗃️ Representado por **lista** ou **matriz** de adjacência
- ✅ Ideal para modelar **redes e relacionamentos complexos**
